# Code of Conduct

This project welcomes technical discussion that is direct, respectful, and focused on improving the software.

## Expected Behavior

- Treat contributors and users with respect.
- Critique code, design, and evidence rather than people.
- Keep discussions practical and relevant to the project.
- Protect private data. Do not post raw Claude Code exports, secrets, private file paths, or unredacted logs.
- Give maintainers enough context to reproduce issues without exposing sensitive information.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing another person's private information.
- Sharing exploit details publicly when a private security channel is available.
- Repeatedly derailing issues, pull requests, or reviews after maintainers ask you to stop.

## Enforcement

Maintainers may edit or remove inappropriate content, close issues or pull requests, and restrict participation when needed to protect contributors or users.

For conduct concerns, contact a maintainer through the repository's listed contact path. If no private contact path is available, open a brief issue asking for one and avoid including sensitive details.
